Project Overview

Notify is a modern PSD landing page template perfect to present your mobile app. The PSD file is fully layered and editable. This resource (PSD) has been created by Michael Reimer from Best Psd Freebies.

Project details

  • Responsive web page, design responds well from mobile (320px), to tablet and desktop screen sizes.

  • Header with app name, links to major mobile app markets.

  • Get in touch section, social network links.

  • Call-to-action email subscription.

  • Embedded YouTube video.

  • Flat design.

  • Retrive testimonials without reloading the page - AJAX.

  • Animated interactions.
